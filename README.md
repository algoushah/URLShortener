# Ayoub URL Shortener

A modern, fast, and easy-to-use URL shortener web application. Transform long URLs into short, shareable links instantly.

## ✨ Features

- **Instant Shortening** - Convert long URLs to short links in seconds
- **No Registration Required** - Start shortening immediately
- **Permanent Links** - Shortened URLs never expire
- **One-Click Copy** - Copy shortened URLs to clipboard instantly
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Clean Interface** - Modern terminal-inspired design
- **Free to Use** - Powered by TinyURL API

## 📖 How to Use

1. Open the application in your web browser
2. Paste your long URL into the input field
3. Click "Execute Shortening Protocol" button
4. Copy your shortened URL using the copy button
5. Share your short link anywhere!

## 🛠️ Installation

### Option 1: Use GitHub Pages (Recommended)

1. Fork this repository
2. Go to Settings → Pages
3. Enable GitHub Pages from `main` branch
4. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Run Locally

1. Clone the repository:
```bash
git clone https://github.com/yourusername/ayoub-url-shortener.git
```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.
